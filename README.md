# Personal Website

**Personal Website Application for Project Showcase and Blogging**

working demo:
**[dorletz](http://dorletz.com)**

---

### _To run:_

```
    npm i
    npm run dev
```

---

### _Tech stack:_

- React
- Typescript
- Redux
- Axios
- Bootstrap
- Sass
- Nodejs
- MikroORM
- MySql
